# Блог-платформа на Django

Веб-приложение — полнофункциональная блог-платформа с системой пользователей, публикаций, комментариев и гибким управлением правами доступа. Реализована с использованием Class-Based Views (CBV), Function-Based Views (FBV) и стандартной архитектуры Django.

# Основные возможности:

Полная система аутентификации: регистрация, вход/выход, смена пароля.

Создание, редактирование, удаление постов с опцией загрузки изображений и возможностью оставлять комментарии.

Автор поста/комментария имеет исключительные права на его редактирование и удаление.

Поддержка отложенных и скрытых публикаций.

Пагинация постов для удобства навигации.

Настроен файловый бэкенд для тестирования отправки email (например, для сброса пароля).

Кастомные страницы ошибок (404, 403csrf, 500) и обновленные статические страницы.

# Технологии:

Backend: Python 3.10, Django, Django ORM

База данных: SQLite

Frontend: HTML, CSS (шаблоны Django)

Система контроля версий: Git

# Установка и запуск:

Предварительные требования:
Убедитесь, что у вас установлен Python (версии 3.8 или выше).

1. Клонирование репозитория:

git clone <https://github.com/ninnafil/blog-platform.git>
cd django_sprint4

2. Создание и активация виртуального окружения:

python -m venv venv
Для Windows: venv\Scripts\activate
Для macOS/Linux: source venv/bin/activate

3. Установка зависимостей:

pip install -r requirements.txt

4. Применение миграций базы данных:

python manage.py migrate

5. Создание суперпользователя (для доступа к админ-панели):

python manage.py createsuperuser

6. Запуск сервера разработки:

python manage.py runserver

7. Открытие в браузере:

Перейдите по адресу: http://127.0.0.1:8000
Админ-панель: http://127.0.0.1:8000/admin (используйте логин/пароль суперпользователя)
